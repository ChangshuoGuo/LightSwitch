# LightSwitch
疫情期间使用手头单片机及模块制作的开关灯工具。

## V1.0
使用 *Arduino UNO R3开发板* + *SG90舵机* + *VS1838B红外接收模块*。

用红外遥控器发送指令，控制舵机转动特定角度，拨动开关实现遥控灯开关功能。

## V1.1
增加蓝牙模块功能，可通过手机连接模块发送指令控制开关。

优化红外接收部分代码。
